dart:ffi library

Foreign Function Interface for interoperability with the C programming language.

NOTE: Dart:FFI is in technical preview. The overall feature is incomplete, may contain issues, and breaking API changes are still expected.

For further details, please see: https://dart.dev/server/c-interop
